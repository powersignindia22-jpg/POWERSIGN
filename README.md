# POWERSIGN INDIA (P) LTD

Premium solar engineering and lithium energy storage solutions for Kerala's premier properties.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Performance**: Static Site Generation (Pre-rendering) via `vite-plugin-prerender`

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Features

- **SSG Ready**: Optimized for SEO with pre-rendered HTML for all routes.
- **Dynamic Testimonials**: Responsive carousel with auto-play.
- **Project Gallery**: Masonry grid showcasing real-world installation landmarks.
- **Centralized Data**: Easy content management via `src/data/`.
